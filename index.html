Apa itu Siventalis?

Siventalis adalah sistem informasi inventarisir alat dan fasilitas politeknik negeri ketapang berbasis website yang digunakan untuk melakukan pengelolaan data alat dan fasilitas politeknik negeri ketapang.

BEBERAPA KEUNGULAN SIVENTALIS

  • Mudah Diakses
  • Mudah, cepat dan akurat dalam menyimpan data
  • Fleksibel

Fitur Aplikasi Siventalis

Rekap Data

Fitur ini dapat menampilkan seluruh data inventaris yang ada pada setiap ruangan dan prodi

Barcode

Fitur ini dapat digunakan untuk scan barcode yang kemudian menampilkan detail inventaris

Cetak Barcode

Fitur ini dapat digunakan untuk mencetak barcode inventaris

Menu Data Prodi

Menu ini digunakan untuk melakukan pengelolaan data prodi yang ada di politeknik negeri ketapang, pada menu ini admin dapat menambahkan data prodi, mengedit data prodi, serta menghapus data prodi.

Menu Data Ruangan

Menu ini digunakan untuk melakukan pengelolaan data ruangan pada setiap prodi yang ada di politeknik negeri ketapang, pada menu ini admin dapat menambahkan data ruangan, mengedit data ruangan, melihat detail data ruangan, serta menghapus data ruangan.

Menu Data Inventaris

Menu ini digunakan untuk melakukan pengelolaan data inventaris yang ada pada setiap prodi politeknik negeri ketapang, pada menu ini admin dapat menambahkan data inventaris, mengedit data inventaris, melihat data inventaris, serta menghapus data inventaris.

Menu Rekap Data

Menu ini dapat digunakan untuk merekap data inventaris yang ada di setiap ruangan dan setiap prodi, pada menu ini admin juga dapat melakukan cetak barcode.
